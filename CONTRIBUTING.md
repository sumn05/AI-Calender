# 🤝 기여 가이드

## 코드 스타일

### JavaScript
- ES6+ 문법 사용
- 변수명은 camelCase
- 함수명은 동사로 시작 (render, create, handle 등)
- 주석은 필요한 곳에만 간결하게

### CSS
- 클래스명은 kebab-case
- 컴포넌트별로 주석 구분
- 중첩 깊이 최소화

### HTML
- 시맨틱 태그 사용
- 접근성을 고려한 마크업
- id는 JavaScript에서만 사용

## 파일 구조 규칙

### 현재 구조 유지
- `index.html`: 메인 페이지
- `detail.html`: 상세보기 페이지
- `script.js`: 메인 로직 (코드 변경 없음)
- `detail.js`: 상세 페이지 로직 (코드 변경 없음)
- `styles.css`: 전체 스타일 (코드 변경 없음)

### 향후 개선 시
- 기능별 모듈화 고려
- 컴포넌트 단위 분리
- 공통 유틸리티 분리
