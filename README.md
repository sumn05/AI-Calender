# AI 캘린더

## 프로젝트 개요

본 프로젝트는 소프트웨어 개발 및 팀 프로젝트 진행 시 필요한 다양한 도구(Notion, 카카오톡, Visual Studio Code, Figma 등)를 효율적으로 관리하고 접근할 수 있는 통합 플랫폼을 구현한 웹 애플리케이션입니다.

기존에는 각 도구별로 분산되어 있어 사용성과 접근성 측면에서 불편함이 있었으며, 이러한 문제를 해결하기 위해 본 애플리케이션을 개발하게 되었습니다.

## 핵심 기능

### 1. 캘린더 기반 일정 관리
- 월별 캘린더 뷰 제공
- 날짜별 이벤트 생성, 수정, 삭제 기능
- 각 이벤트에 대한 상세 정보 관리

### 2. 통합 링크 관리 시스템
- 다양한 개발 도구 및 협업 서비스에 대한 빠른 접근을 위한 링크 저장소
- 주요 서비스 아이콘 지원 (Notion, 카카오톡, VS Code, Figma 등)
- 캘린더 이벤트에 직접 링크 연동 가능
- 별도의 캘린더 선택 없이 필요한 도구에 즉시 접근 가능한 독립 링크 등록 기능

### 3. 상세보기 페이지
- Notion 스타일의 자유로운 레이아웃 구현
- 메모 작성 및 관리
- 이미지 업로드 및 관리
- 링크 통합 관리

### 4. AI 기반 음성 녹음 요약
- 팀 미팅 및 회의 내용 실시간 녹음
- Google Gemini API를 활용한 자동 요약 생성
- Web Speech API 및 MediaRecorder API를 통한 음성 인식 및 텍스트 변환

### 5. 운세 확인 기능
- 사용자 생년월일 및 별자리 기반 일일 운세 제공
- 운세 점수 산정 및 종합 운세 분석
- 행운의 숫자 및 키워드 추천

### 6. 동적 레이아웃 커스터마이징
- 메모, 녹음, 사진 카드의 자유로운 배치 및 위치 조정
- 드래그 앤 드롭 인터페이스를 통한 사용자 경험 최적화
- 카드 크기 조절 기능 (4개 모서리 리사이징)
- 카드 확장/축소 기능
- 레이아웃 상태의 자동 저장 및 복원

## 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **데이터 저장**: LocalStorage API
- **AI 서비스**: Google Gemini API
- **브라우저 API**: 
  - Web Speech API (음성 인식)
  - MediaRecorder API (오디오 녹음)

## 설치 및 실행

1. 저장소 클론:
```bash
git clone https://github.com/sumn05/AI-Calender.git
```

2. 프로젝트 디렉토리로 이동:
```bash
cd AI-Calender
```

3. 웹 브라우저를 통해 `index.html` 파일을 실행합니다.

## 기능 사용 가이드

### 이벤트 관리
1. 캘린더에서 원하는 날짜를 선택합니다.
2. "+ 새로운 이벤트" 버튼을 클릭하여 모달을 엽니다.
3. 이벤트 제목 및 내용을 입력합니다.
4. 필요 시 관련 링크를 추가합니다.
5. 저장 버튼을 클릭하여 데이터를 저장합니다.

### 상세보기 페이지
1. 이벤트 항목의 "상세보기" 버튼을 클릭합니다.
2. 새로운 탭에서 상세보기 페이지가 열립니다.
3. 메모, 사진, 링크를 추가하거나 수정할 수 있습니다.
4. 카드를 드래그하여 위치를 변경하거나 모서리를 조절하여 크기를 변경할 수 있습니다.

### 음성 녹음 및 AI 요약
1. 상세보기 페이지에서 "🎤 녹음 시작" 버튼을 클릭합니다.
2. 브라우저의 마이크 권한을 허용합니다.
3. 녹음을 완료한 후 "⏹ 녹음 중지" 버튼을 클릭합니다.
4. "🤖 AI 요약하기" 버튼을 클릭하여 Google Gemini API를 통해 자동 요약을 생성합니다.

### 운세 확인
1. 메인 페이지에서 "🔮 운세 추가하기" 버튼을 클릭합니다.
2. 생년월일과 별자리를 입력합니다.
3. "운세 확인하기" 버튼을 클릭하여 결과를 확인합니다.

### 링크 추가
1. 우측 하단의 "+" 버튼을 클릭합니다.
2. 서비스를 드롭다운에서 선택하거나 직접 입력합니다.
3. 해당 서비스의 URL을 입력합니다.
4. 저장 버튼을 클릭하여 링크를 등록합니다.

## 기술적 고려사항

- **브라우저 호환성**: 최신 버전의 Chrome, Edge, Firefox 등의 브라우저 사용을 권장합니다.
- **마이크 권한**: 음성 녹음 기능 사용을 위해서는 브라우저의 마이크 접근 권한이 필요합니다.
- **API 키 설정**: AI 요약 기능 사용 시 Google Gemini API 키가 필요하며, 상세보기 페이지에서 설정할 수 있습니다.
- **데이터 저장 방식**: 모든 데이터는 클라이언트 측 LocalStorage에 저장되며, 브라우저별로 독립적으로 관리됩니다.
