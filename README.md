# AI 캘린더

소프트웨어 개발뿐만 아니라 팀 프로젝트 때 Notion, 카카오톡, VS Code, Figma 등 여러 웹과 앱을 사용해야 하기 때문에 사용에 불편함을 느꼈고, 이것을 해결하기 위해 좋은 앱이 없을까 하는 생각에 만들게 되었습니다.

이 웹은 캘린더 기능을 통해서 각 날짜별로 무엇을 할지 저장할 수 있고, 캘린더에 Notion이나 카카오톡처럼 바로 갈 수 있는 링크를 추가했습니다. 또 굳이 캘린더를 선택하지 않아도 바로 자신이 필요한 것들을 사용할 수 있게 링크를 등록할 수 있는 창을 만들었고, 또 캘린더에 일정을 등록하는 것뿐만 아니라 일정을 자세히 작성할 수 있게 메모와 녹음, 사진, 링크 웹을 하나 더 만들었고, 또 팀플 때 녹음을 통해 녹음 기능은 AI를 활용하여 요약을 할 수 있습니다. 또 메모와 녹음, 사진은 동적으로 사용자가 배치할 수 있고 크기도 마음대로 조절할 수 있게 배치하였고 사용자 편의 기능도 도왔습니다.

## 주요 기능

### 📅 캘린더 관리
- 월별 캘린더 뷰
- 날짜별 이벤트 추가/수정/삭제
- 이벤트 상세보기 기능

### 🔗 링크 관리
- 빠른 접근을 위한 링크 저장소
- 다양한 서비스 아이콘 지원 (Notion, 카카오톡, VS Code, Figma 등)
- 캘린더 이벤트에 링크 추가 가능

### 📝 상세보기 페이지
- Notion 스타일의 자유로운 레이아웃
- 메모 작성 기능
- 사진 업로드 및 관리
- 링크 관리

### 🎤 AI 기반 요약
- 팀 미팅 녹음 내용을 AI가 자동으로 요약
- Google Gemini API 활용
- 음성 인식 및 텍스트 변환

### 🔮 운세 확인
- 생년월일과 별자리를 바탕으로 한 일일 운세 확인 기능
- 오늘의 운세 점수 및 종합 운세 제공
- 행운의 숫자 및 키워드 제공

### 🎨 커스터마이징 가능
- 메모, 녹음, 사진 카드를 자유롭게 배치하고 크기 조절
- 드래그 앤 드롭으로 카드 위치 이동
- 카드 확장/축소 기능
- 레이아웃 자동 저장

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Storage**: LocalStorage
- **API**: Google Gemini API
- **Browser APIs**: Web Speech API, MediaRecorder API

## 실행 방법

1. 저장소를 클론합니다:
```bash
git clone https://github.com/sumn05/AI-Calender.git
```

2. 프로젝트 디렉토리로 이동합니다:
```bash
cd AI-Calender
```

3. `index.html` 파일을 웹 브라우저로 엽니다.

4. 전화번호를 입력하여 로그인합니다.

## 사용 방법

### 이벤트 추가
1. 캘린더에서 날짜를 선택합니다.
2. "+ 새로운 이벤트" 버튼을 클릭합니다.
3. 이벤트 제목과 내용을 입력합니다.
4. 필요시 링크를 추가합니다.
5. 저장 버튼을 클릭합니다.

### 상세보기
1. 이벤트 항목에서 "상세보기" 버튼을 클릭합니다.
2. 새 탭에서 상세 페이지가 열립니다.
3. 메모, 사진, 링크를 추가하거나 수정할 수 있습니다.
4. 카드를 드래그하여 위치를 변경하거나 크기를 조절할 수 있습니다.

### 녹음 및 AI 요약
1. 상세보기 페이지에서 "🎤 녹음 시작" 버튼을 클릭합니다.
2. 마이크 권한을 허용합니다.
3. 녹음이 완료되면 "⏹ 녹음 중지" 버튼을 클릭합니다.
4. "🤖 AI 요약하기" 버튼을 클릭하여 자동 요약을 생성합니다.

### 운세 확인
1. 메인 페이지에서 "🔮 운세 추가하기" 버튼을 클릭합니다.
2. 생년월일과 별자리를 입력합니다.
3. "운세 확인하기" 버튼을 클릭합니다.
4. 오늘의 운세를 확인합니다.

### 링크 추가
1. 우측 하단의 "+" 버튼을 클릭합니다.
2. 서비스를 선택하거나 직접 입력합니다.
3. 링크 URL을 입력합니다.
4. 저장 버튼을 클릭합니다.

## 참고사항

- **브라우저 제한**: 최신 브라우저 사용을 권장합니다 (Chrome, Edge, Firefox 등)
- **마이크 권한**: 녹음 기능 사용 시 브라우저의 마이크 권한이 필요합니다.
- **API 키**: AI 요약 기능 사용 시 Google Gemini API 키가 필요합니다. 상세보기 페이지에서 설정할 수 있습니다.
- **데이터 저장**: 모든 데이터는 LocalStorage에 저장되며, 브라우저별로 독립적으로 관리됩니다.
- **전화번호 로그인**: 전화번호는 LocalStorage에 저장되며, 별도의 인증 없이 사용됩니다.
